# BridgeLabz SIPP Training 2026

This repository contains Java practice work completed as part of the BridgeLabz SIPP Training 2026.

## Completed Work

The repository currently includes **3 completed scenario-based practice exercises** under `collections-practice/scenario-based-ques`:

1. **Event Entry Verification System**
   - Console-based application for registering participants
   - Prevents duplicate registrations
   - Displays all registered participants
   - Shows total attendee count

2. **Smart Classroom Attendance Tracker**
   - Attendance tracking application for classroom scenarios
   - Includes core tracker logic and app entry point

3. **Smart Parking Slot Manager**
   - Simple vehicle parking management practice
   - Handles entry and exit operations using a list
